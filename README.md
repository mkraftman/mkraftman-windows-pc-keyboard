# Mkraftman Windows PC Keyboard

Custom Lovelace card for Home Assistant providing always-active keyboard input for a Windows PC via Unified Remote.

Modelled after the Apple TV keyboard card but permanently in active mode.

## Features
- Always-visible keyboard input (no toggle needed)
- Opens native soft keyboard on tap
- Sends individual keystrokes via Unified Remote
- Handles special keys (Backspace, Enter, Tab, Escape)

## Installation
Install via HACS as a custom repository.

## Requirements
- [hass-unified-remote](https://github.com/akshansh1998/hass-unified-remote) integration
- Unified Remote Server running on target Windows PC
